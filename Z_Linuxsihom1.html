  • 묶음 특정 경로로 복사 시키기(이거 실패중)
    • sudo mkdir /backup/
    cp master.tar.gz /backup/
    cd /backup/
    ls (master.tar.gz나옴)
    cd /home/
    ls (lost + found나옴)
    cp /backup/master.tar.gz ./
    tar zxvf master.tar.gz (파일을 풀어주다)
    ls (lost+found, master, master.tar.gz)
    mount (/dev/sdb1 on /home type ext4 같은 거 뜨면 됨)
