#### 통신용량 

```
		       S

```

#### 대역폭 계산법 
fupper(가장 큰 주파수) - flower(가장작은주파수) = w(대역폭)  
 
#### bps 구하기 예시 
대역폭 w : 50hg   
S앤 : 15[db] 라면  

C = 50 log2(1+15)  
= 50 log2 16  
(16을 2^4로 만든 것)  
= 50 log2  2^4  
= 4 * 50 log2 ^2  
= 4 * 50 * 1 ^1  
= 4 * 50  
= 200 [bps]  

중간 제곱표현이 해깔리네  
일단 저기 log2 의 2 부분은 밑수 말하는 거임  
밑수 2 제곱 2임, 그래서 결과가 저럼

#### RZ MRZ 
RZ: Return to Zero(0으로 되돌아온다)  
```
+1  ㅁ        ㅁ ㅁ             
  0  --ㅁ--ㅁ- ㅁ--ㅁ--ㅁ---ㅁ---ㅁ------ 
 -1       ㅁ           ㅁ     ㅁ   ㅁ   ㅁ
```
이런 기준 점에서 0이면 올라가고 1이면 내려가는  
그래프가 있다고 치면, 올라갔다 내려가기 전에  
중간중간 0으로 선이 오는 방식이고  

NRZ: Non Return to Zero(0으로 되돌아오지 않는다)  
``` 
+1  ㅁㅁ       ㅁㅁㅁ     ㅁ ㅁ     ㅁ   
  0  --------------------------------- 
 -1       ㅁㅁ         ㅁㅁ ㅁ  ㅁㅁ
```
반대로는 그냥 0 1 신호에 따라  
+1 -1 중 하나에만 그래프가 가있음
