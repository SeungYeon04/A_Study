###  객체로 값 전달 + 클래스 안 메소드 출력 
```
class Rabbit :
  shape = ""
  xpos=0
  xpos=0

  def goto(self,x,y):
    self.xpos=x
    self.ypos=y
    print(x,y)
    # 생성자X 소멸자X (모두 No정의) 

r1 = Rabbit()
r1.goto(100,100)

# 100 100 

#<- 스탯과 힙 중 객체는 힙에 들어가 존재  
```

### __del__(소멸자) 메소드 - 프로그램 죽으면(끝나면) 출력
```
class Rabbit :
  shape = ""
  xpos=0
  xpos=0

  def goto(self,x,y):
    self.xpos=x
    self.ypos=y
    print(x,y)
    # 생성자X 소멸자X (모두 No정의) 

  def __del__(self): # 프로그램 죽으면(끝나면) 출력 
    print("나 죽어!!!")

r1 = Rabbit()
r1.goto(100,100)

# 100 100 
# 나 죽어!!!

# 클래스다이어그램: 먹이클래스 유통기한멤버변수 메서드 등 설계 계획 사각형 *모델링* 어캐 만들지 
```
