### 3단씩 묶어서 "3중 for문 써서" 구구단 출력하게 하기 
```
package chaper01;

public class Task3_05 {

	public static void main(String[] args) {
		
		int k = 0; int i = 0; 
		
		for(i = 1; i <= 9; i+=3) { 
			
			System.out.println(i+"단\t"+(i+1)+"단\t"+(i+2)+"단\t"); 
			 
			for(int j = 1; j <= 9; j++)
			{
				for(k = 0; k < 3; k++) {
					System.out.print((k+i)+"*"+j+"="+(j*(k+i))+"  ");	
				} k = 0; 
				System.out.println();		
			} 
			System.out.println();	
		}
	}
}
```
