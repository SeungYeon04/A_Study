smbpasswd -a master (삼바 패스워드 걸기. 안 하면 랜섬웨어도 위험 공부니 = 123456으로 걸기)

컴퓨터 > 속성 > 작업그룹 보면 WORKGROUP 똑같이 되어있음
변경 필요X


> sudo nano /etc/samba/smb.conf

(거의 맨 위에 통으로 추가)

   writable = yes 
   guest ok = no 
   create mode = 0777 
   directory mode = 0777 
   valid users = @sambaGreup
