  • 110111 변환

    • 2진수 -> 16진수
    (추가:00)110111 -> 0011(3)/0111(7)-> (37)16

    2진수 -> 8진수
    110111 -> 110(6)/111(7) -> (67)8


    16진수는 4개씩 나눔
    예시: (추가:0)100(4)/1001(9).0100(4)

    8진수는 3개씩 나눔
    예시: 001(1)/001(1)/001(1).001(1)/0(x)


  • 2의 보수와 1의 보수 뜻

    2진수(문제1) = 10101
    1의 보수 = 01010
    2의 보수 = 01011

    1의 보수는 1은 0으로 0은 1로 바꾸면 되고
    2의 보수는 1의 보수 값에서
    맨 최하위(LSB)에 1을 더 더하면 된다.
